# Java Task Manager

Simple console-based task manager application written in Java.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks

## Architecture
The project is structured into multiple layers:
- UI layer (Console interaction)
- Service layer (business logic)
- Model layer (data representation)

## Technologies
- Java
- OOP (encapsulation, classes, methods)

## Future Improvements
- Database integration (MySQL)
- REST API (Spring Boot)
- Web interface

## How to run
1. Clone the repository
2. Open in IntelliJ IDEA or any Java IDE
3. Run Main class
